# Goodwill Donation Tracker App

This is a code bundle for the Goodwill Donation Tracker App. The original project is available at [https://www.figma.com/design/yeV5CyxKkTSF7My1HahqbM/Donation-Management-App](https://reply-metric-03492223.figma.site).

# Description

The Goodwill Donation Tracker App is designed to take a systemic approach to increase efficiency of Goodwill donations. The digital platform “Goodwill Tracker” includes sections tailored to donors, employees, and Goodwill corporate. Key features in the donors portal include a Donation Checker to ensure donations are usable, clear donation guidelines, and accessibility features. The employee portal features donation image scanning and tracking using an AI to automatically scan items where manual adjustments are permissible. The corporate portal includes key metrics and analytics that reflect long term trends over time. The goal of this project is to solve the Track 3 Goodwill Industries Problem Statement 4 at the [George Hacks 2026 Hackathon](https://www.georgehacks.org/#tracks). 

# Running the code

Run `npm i` to install the dependencies.
Run `npm run dev` to start the development server.
