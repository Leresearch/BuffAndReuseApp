# Goodwill Donation Tracker App

This is a code bundle for the Goodwill Donation Tracker App. The original project is available at https://www.figma.com/design/yeV5CyxKkTSF7My1HahqbM/Donation-Management-App.

# Description

The Goodwill Donation Tracker App is designed to take a systemic approach to increase efficiency of Goodwill donations. The Digital platform “Goodwill Tracker” includes sections tailored to Donors, Employees, and Goodwill Corporate. The donors portal key features includes a Donation checker to ensure donations are usable, clear donation guidelines, and accessibility features. The employee portal key features includes donation image scanning and tracking using an AI to automatically scan items where manual adjustments were permissible. The corporate portal key features includes key metrics and analytrics and refelct long term trends over time. The goal of this project is to solve the Track 3 Goodwill Industries Problem Statement 4 at the George Hacks 2026 Hackathon. 

# Running the code

Run `npm i` to install the dependencies.
Run `npm run dev` to start the development server.
